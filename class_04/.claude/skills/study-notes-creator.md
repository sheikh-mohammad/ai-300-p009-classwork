# Study Notes Creator Skill

This skill generates comprehensive study notes for a particular lesson with key points for reinforcement and revision.

## Purpose

The Study Notes Creator skill helps students and learners create well-structured study materials that follow best practices for educational reinforcement. The generated notes include all the essential components needed for effective review and retention.

## Features

- **Summary at the top**: Each set of notes begins with a concise summary of the main concepts
- **Key terms highlighted**: Important terminology is clearly defined and emphasized
- **Questions at the end**: Review questions are provided to test understanding
- **Easy to read formatting**: The notes are structured in a clear, organized format

## Usage

To use this skill, provide the following information:

- `lesson_topic`: The subject or topic of the lesson
- `lesson_content` (optional): The content to base the notes on
- `key_points` (optional): Specific points you want to emphasize

## Output Format

The skill generates notes in Markdown format with:

1. A clear title
2. A summary section
3. Key terms and definitions
4. Main content section
5. Review questions
6. Additional notes section

## Benefits

- Structured format supports effective studying
- Questions help reinforce learning
- Clear organization improves comprehension
- Consistent format across different topics
