## Markdown Learning Quiz

جب بھی یہ فائل کھولی جائے تو یہ سوالات پوچھیں:

### سوال 1: Headings
**سوال:** Markdown میں سب سے بڑی heading کیسے بناتے ہیں؟
- A) `# Heading`
- B) `## Heading`
- C) `### Heading`
- D) `#### Heading`

**جواب:** A) `# Heading`

---

### سوال 2: Bold اور Italic
**سوال:** اگر آپ کو کوئی لفظ **bold اور italic** دونوں بنانے ہوں تو کیا syntax استعمال کریں گے؟
- A) `*text*`
- B) `**text**`
- C) `***text***`
- D) `__text__`

**جواب:** C) `***text***`

---

### سوال 3: Lists
**سوال:** Unordered list میں کون سی علامت استعمال ہوتی ہے؟
- A) `1.`
- B) `-` یا `*`
- C) `>`
- D) `#`

**جواب:** B) `-` یا `*`

---

### سوال 4: Links
**سوال:** Markdown میں link کیسے بناتے ہیں؟
- A) `[text](url)`
- B) `(text)[url]`
- C) `<text|url>`
- D) `{text:url}`

**جواب:** A) `[text](url)`

---

### سوال 5: Code Blocks
**سوال:** Inline code کے لیے کیا استعمال ہوتا ہے؟
- A) ```` ``` ````
- B) `` ` ``
- C) `---`
- D) `***`

**جواب:** B) `` ` ``

---

### سوال 6: Tables
**سوال:** Markdown میں table کی header row کے بعد کیا لکھتے ہیں؟
- A) `---`
- B) `| --- | --- |`
- C) `===`
- D) `***`

**جواب:** B) `| --- | --- |`

---

### سوال 7: Blockquotes
**سوال:** Blockquote کے لیے کون سی علامت استعمال ہوتی ہے؟
- A) `>`
- B) `|`
- C) `-`
- D) `*`

**جواب:** A) `>`

---

### سوال 8: Images
**سوال:** Markdown میں image کیسے شامل کرتے ہیں؟
- A) `[image](url)`
- B) `![alt text](url)`
- C) `<img src="url">`
- D) `{image:url}`

**جواب:** B) `![alt text](url)`

---
