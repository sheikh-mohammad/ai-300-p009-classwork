# Example Usage: Study Notes Creator

## Basic Usage

To create study notes for a lesson, use the skill with the lesson topic:

```
/study-notes-creator lesson_topic="Introduction to Machine Learning"
```

## Advanced Usage

To create study notes with specific content:

```
/study-notes-creator lesson_topic="Photosynthesis" lesson_content="Photosynthesis is the process by which green plants and some other organisms use sunlight to synthesize foods with the help of chlorophyll. The process converts light energy into chemical energy stored in glucose molecules." key_points="light energy, chlorophyll, glucose, carbon dioxide, oxygen"
```

## Sample Output Structure

The skill will generate notes in this format:

# Study Notes: [Lesson Topic]

## Summary
[Brief overview of the main concepts]

## Key Terms and Concepts
- **Term 1**: Definition
- **Term 2**: Definition

## Main Content
[Detailed information about the topic]

## Questions for Review
1. [Question about main concept]
2. [Question about key terms]
3. [Application question]

## Additional Notes
[Any supplementary information]
