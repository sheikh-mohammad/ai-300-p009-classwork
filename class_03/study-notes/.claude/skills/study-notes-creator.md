# Study Notes Creator Skill

## Purpose
Generate comprehensive study notes for academic lessons designed for reinforcement and revision. This skill produces well-structured, pedagogically sound notes that facilitate learning retention and conceptual understanding.

## Invocation
```
/study-notes-creator [lesson topic]
```

## Output Structure

All generated study notes shall adhere to the following standardized format:

### 1. Summary Section
- Provide a concise executive summary (150-250 words) at the top of the document
- Synthesize the primary concepts and learning objectives
- Establish the foundational context for the lesson

### 2. Key Terms and Definitions
- Highlight and define essential terminology using **bold formatting**
- Present definitions in clear, academic language
- Organize terms in logical conceptual groupings
- Include context for each term's application

### 3. Main Content
- Organize material into logical sections with clear hierarchical structure
- Use numbered lists and bullet points for readability
- Provide explanations with appropriate depth for academic study
- Include relevant examples and applications where applicable

### 4. Review Questions
- Generate 5-8 comprehension and critical thinking questions at the end
- Include a mix of question types:
  - Definitional questions (testing recall)
  - Conceptual questions (testing understanding)
  - Application questions (testing synthesis)
- Provide answer guidance or key points for each question

## Formatting Guidelines

- Utilize Markdown formatting for optimal readability
- Employ headers (##, ###) for clear section delineation
- Use **bold** for key terms and important concepts
- Implement bullet points and numbered lists for clarity
- Maintain consistent spacing and visual hierarchy
- Ensure academic tone throughout all content

## Content Principles

- Maintain scholarly rigor and accuracy
- Present information in a logical, progressive manner
- Balance comprehensiveness with conciseness
- Support claims with evidence-based explanations
- Facilitate active learning through structured review questions
